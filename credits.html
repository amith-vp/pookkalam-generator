Credits

Friends of Figma Kochi

After deciding to create a Pookkalam generator, I found a Figma project Play with Pookalam by Friends of Figma Kochi, A big thanks to FoF Kochi for the inspiration and the SVGs. Check out FoF Kochi.

Random Flower Generator

The random flower generation feature was adapted from an impressive CodePen project by Kevin Levron. See the original flower generator.

Colour Fill Tool

Implementing an efficient color fill tool was challenging. After experimenting with various flood fill algorithms, This solution proved to be the most effective for the need. Github .

Curve Tool

The curve tool, essential for creating intricate patterns, was inspired by a clever implementation in a CodePen by Rory Duncan. Original curve tool.

Dyuthi font

The Dyuthi font used in this project was designed by Hiran Venugopalan. You can find more about the font smc.org.in/fonts/dyuthi.
